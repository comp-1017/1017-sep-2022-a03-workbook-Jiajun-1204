Create navigation links within a web site

There are several techniques to navigation within a web site. The semantic tag for navigation is the <nav> tag; You can nevigation through your web site by using relative addressing. Four our examples we will use the unordered list tag (<vl>).

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a spectific place on the current page (on page navigation)

Navigational menus are place in your <header> Semantic area. They may also be place in your <footer> area as a sitemap. The nav tag like other semantic tags should have a <hx> tag. However this header is optional. If you omit the header tag you will receive an document outline error: untitled Section. Until you are shown how to avoid this error message, it will be acceptable on your document outline.
